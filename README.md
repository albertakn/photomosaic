# photomosaic

Собирает фотомозаику — портрет (лицо), выложенный из множества маленьких
фоток-«плиток», скачанных из публичного Instagram-аккаунта.

## Установка

```bash
uv sync
```

## 1. Скачать фотки (плитки)

Анонимно, без логина — работает только с **публичными** аккаунтами:

```bash
uv run main.py scrape --user https://instagram.com/имя_аккаунта/
```

Картинки складываются в `tiles/`. Instagram ограничивает анонимный доступ:
если упрётся в лимит — подожди несколько минут и запусти команду снова, уже
скачанное сохранится.

## 2. Собрать мозаику

```bash
uv run main.py build --target face.jpg --tiles tiles --out mosaic.png
```

Целевое фото (`--target`) лучше заранее кадрировать в портрет.

### Параметры подбора качества

| Флаг          | По умолч. | Что делает                                            |
|---------------|-----------|-------------------------------------------------------|
| `--cols`      | 80        | ячеек по ширине; больше → детальнее лицо, мельче фотки |
| `--tile-px`   | 50        | размер плитки в пикселях (итоговое разрешение)         |
| `--overlay`   | 0.15      | подмес цвета 0..1; меньше → сильнее видно фотки        |
| `--no-repeat` | 3         | радиус (в рядах) запрета повтора одной плитки          |

Отпечатки плиток кэшируются в `tiles_cache.npz` — повторные сборки быстрее.

## Как это устроено

1. Целевое фото бьётся на сетку `cols × rows`.
2. У каждой плитки и каждой ячейки считается 3×3 отпечаток в цветовом
   пространстве **Lab** (27 чисел) — учитывается не только тон, но и структура.
3. К каждой ячейке подбирается ближайшая по отпечатку плитка.
4. Соседние ячейки не используют одну плитку (антиповтор).
5. Сверху лёгкий цветовой подмес (`--overlay`), чтобы лицо читалось.
